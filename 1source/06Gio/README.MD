# Gio

## Running Gio

### Windows

* [Download] and install Go
* Install all the dependencies: `go install gioui.org/cmd/gogio@latest`
* Navigate to sample project: `cd ...\1source\6Gio\00SampleProject`
* Update all the dependencies: `go mod tidy`
* Run the program: `go run .`
